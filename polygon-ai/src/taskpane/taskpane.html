Custom Functions

getTickerDetails(ticker, [property])

Retrieves details for a specific stock ticker from Polygon.io.

Parameters:

  • ticker: The stock ticker symbol (e.g., "AAPL").
  • property: (Optional) The specific property to retrieve (e.g., "name", "market"). If omitted, returns a formatted string.

Examples:

  • =getTickerDetails("AAPL") // Returns "AAPL - Apple Inc. (stocks)"
  • =getTickerDetails("AAPL", "name") // Returns "Apple Inc."

Note: Ensure you have set your Polygon.io API key above.
